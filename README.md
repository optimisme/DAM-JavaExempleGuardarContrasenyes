# Exemple contrasenyes en Java #

En aquest projecte hi ha un exemple de com ocultar contrasenyes per tal que siguin ininteligibles

### Compilació i funcionament ###

A Linux i OSX:

```
./compile.sh
```

A Windows Powershell:

```
.\compile.bat
```
